Конфигурация оборудования

У стримера ограниченные физические возможности, поэтому все оборудование подбиралось с учетом возможности комфортного управления с компьютера с помощью мышки и голоса.

Апгрейдные планы

  • Видеокарта: заменить RTX 5060 -> RTX 5070/5080.
  • SSD: добавить еще один на 1TB.
  • Оперативная память: заменить DDR4 32GB -> DDR5 32GB.
  • Клавиатура: заменить Logitech MK240 Nano -> NuPhy Air60 V2 Aloe Switches.
